# BioPlex-An-Open-Source-Living-Research-System
BioPlex is an open‑source, modular micro‑ecosystem platform for building automated living research habitats. Think of BioPlex as a “bonsai tree” system for cultivating automated micro‑ecosystems for self‑study and experimentation. BioPlex is a Pico‑powered, sensor‑driven habitat framework for creating automated aquatic and terrestrial micro‑biomes.
